

----------An Web Based Application for Area based Restaurant Search--------


What is it?
-----------

This is a site for searching Restaurants which are nearly located according 
to your current location.


How to use?
-----------

User can search Restaurants which are nearly located according to your current 
location. Users can also get much idea about the price of food of selected 
restaurant and also about the exclusive offer of those restaurant if any. 
He/She also find the contact number of choosen restaurant and able to see the 
map whaere the restaurant actually situated. User can submit a review of 
restaurant which is will show the other users.


Users Interaction
-----------------

There are two types of users who can use this. One is Restaurant user and another 
is Person user. Restaurant user can be able to creat it's new profile with its 
real information. It can make offer and set menus of the restaurant by uploading 
jpg or png Image. and person can see it publicly by searching. If person user 
want to rate a restaurant he/she must have to log in with his own authentication. 
User can creat his Id freely and make dynamic ratings of any restaurant. 

Functionality in every page
---------------------------

1. home.php (Main index)

	• Search option (user can search area)
	• Creat Account (restaurant or person can creat own account for login)
	• Login with valid username & password


2. search.php (Page with various restaurant according to search area)

	• Search option (user can search area without going home page again)
	• Left corner user can see Exclusive Restaurants list
	• User can select individual restaurant for details
	• Login and Creat account Still there


3. individual restaurant ( page with details of selected restaurant)

	• Restaurant Profile View
	• Restaurant Details with phon number and address
	• Find location with Google Map
	• Can see menus of restaurant
	• Exclusive offer
	• User can submit review for this restaurant
	• Direct Login, Sign Up and back to Home button


4. Restaurant Profile (restaurant user can modify its own profile)

	• Restaurant Account Information (can update)
	• Offers and Area (can update delete replace offer and area)
	• Update Menu (menu update with image)
	• Password Change (can change Password) 
	• Log out and Home button


5. Personal Account (Person can use his own Account for make ratings)

	• User Account information (updateable)
	• Password Change option
	• Logout and Home button


6. Admin Panel (Maintanance)

	• Different page for admin login
	• Admin can delete both restaurant and personal account permanently
	• Log out and home button


Used tecnique
-------------

This website mainly developed by php (hypertext preprocessor). Used Database SQL, 
Designed using CSS(cascading style sheet) and also html and javascript. Lightbox
for Image viewing.
